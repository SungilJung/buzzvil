# react-native-buzzvil

react native library for buzzvil android sdk

## Installation

```sh
npm install react-native-buzzvil
```

## Usage

```js
import { BuzzvilView } from "react-native-buzzvil";

// ...

<BuzzvilView color="tomato" />
```
